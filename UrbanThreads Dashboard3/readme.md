# 📊 Power BI Dashboard – Order Returns Analysis

## 📌 Project Overview
This project presents a **Power BI dashboard** created to analyze order returns and identify key patterns affecting return rates.  
The dashboard helps understand **why customers return products** and provides insights that can help businesses reduce return rates and improve customer satisfaction.

---

## ✨ Key Features
- 🔹 **Navigation Buttons** to move between different dashboard pages.
- 🔹 **Interactive Sliders (Slicers)** to filter data by **weeks and months**.
- 🔹 **Home Button on Every Page** for quick navigation back to the main dashboard.
- 🔹 **All KPIs in a Single Page** for a quick overview of performance.
- 🔹 **Multiple Visualizations** to analyze order return trends and reasons.
- 🔹 **Clean Formatting & UI Design** to make the dashboard visually appealing.

---

## 🛠 Tools & Technologies Used
- **Power BI**
- **Microsoft Excel**
- **Data Visualization Techniques**
- **Data Cleaning & Transformation**

## 📈 Key Insights
- A significant percentage of returns are caused by **size issues**.
- **Quality defects and wrong items** also contribute to return rates.
- Return patterns change depending on **weekly order volume and promotions**.
- Understanding these factors can help businesses **reduce operational costs and improve product quality**.

---

## 🎯 Conclusion
This dashboard helps businesses **monitor return behavior, identify major return causes, and make data-driven decisions** to reduce return rates and improve customer experience.
